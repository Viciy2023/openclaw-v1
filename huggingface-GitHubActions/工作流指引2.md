## 第一次部署

### ✅ 步骤 1：触发 GitHub Actions 构建

方式 A：推送代码
```bash
cd F:\APK\openclaw-main
git add .
git commit -m "Setup prebuilt Docker image"
git push origin main
```

方式 B：手动触发
1. 进入 GitHub 仓库
2. 点击 Actions 标签
3. 选择 "Build Prebuilt OpenClaw Image"
4. 点击 "Run workflow"
5. 点击绿色的 "Run workflow" 按钮

- [ ] 已触发构建
- [ ] 构建时间：约 60 分钟
- [ ] 构建状态：查看 https://github.com/Viciy2023/openclaw/actions

### ✅ 步骤 2：等待构建完成

### ✅ 步骤 4：配置 HuggingFace Spaces

#### 4.1 创建或更新 Space

- [ ] 进入 https://huggingface.co/spaces
- [ ] 创建新 Space 或进入现有 Space
- [ ] Space 类型：Docker
- [ ] 可见性：Private 或 Public

#### 4.2 上传文件到 HF Space

将以下文件推送到 HF Space 仓库：

- [ ] `Dockerfile-hf` → 重命名为 `Dockerfile`
- [ ] `entrypoint.sh-supabase` → 重命名为 `entrypoint.sh`

```bash
# 在 HF Space 仓库中
cp huggingface-new/Dockerfile-hf Dockerfile
cp huggingface-new/entrypoint.sh-supabase entrypoint.sh
git add Dockerfile entrypoint.sh
git commit -m "Use prebuilt image"
git push
```

#### 4.3 配置 HF Secrets

进入 HF Space → Settings → Repository secrets

- [ ] `SUPABASE_ANON_KEY`：你的 Supabase 匿名密钥
- [ ] `DEPLOY_FILES_BASE_URL`：`https://iwyzcrgcubxdnzjpnhof.supabase.co/storage/v1/object/hf-deploy`

- [ ] 构建时间：5-10 分钟

## 日常更新

### 🔄 更新补丁或配置（最常见）

时间：30秒

1. 修改本地文件
   - [ ] 编辑 `patches/clawmate/*.ts` 或配置文件
   
2. 上传到 Supabase
   - [ ] 覆盖对应的文件
   
3. 重启 HF Space
   - [ ] HF Space → Settings → Factory reboot
   
4. 验证
   - [ ] 检查日志确认文件已下载
   - [ ] 测试功能

### 🔄 更新 entrypoint.sh

时间：10分钟

1. 修改文件
   - [ ] 编辑 `entrypoint.sh-supabase`
   
2. 推送到 HF
   - [ ] 覆盖 `entrypoint.sh`
   - [ ] `git push`
   
3. 等待构建
   - [ ] HF 自动重新构建（10分钟）

### 🔄 更新 OpenClaw 或 openclaw-china

时间：70分钟（60+10）

1. 修改文件
   - [ ] 更新 `openclaw-china-main/`
   - [ ] 或修改 `Dockerfile-prebuilt`
   
2. 触发 GitHub Actions
   - [ ] 推送到 GitHub
   - [ ] 等待构建完成（60分钟）
   
3. 重启 HF Space
   - [ ] Factory reboot
   - [ ] 拉取新镜像（10分钟）

## 故障排查

### ❌ GitHub Actions 构建失败

检查项：
- [ ] Docker Hub Secrets 配置正确
- [ ] `openclaw-china-main` 目录存在
- [ ] 查看 Actions 日志定位错误

### ❌ HF 拉取镜像失败

检查项：
- [ ] 镜像已成功推送到 Docker Hub
- [ ] `Dockerfile-hf` 中的镜像名称正确
- [ ] Docker Hub 镜像是公开的

### ❌ HF 运行时错误

检查项：
- [ ] Supabase 文件已正确上传
- [ ] HF Secrets 配置正确
- [ ] 查看 HF Space Logs
- [ ] 检查 entrypoint.sh 日志输出

### ❌ 视频生成失败

检查项：
- [ ] `video-pipeline.ts` 已上传到 Supabase
- [ ] `plugin.ts` 已上传到 Supabase
- [ ] 环境变量 `HFOPVIDEOSOUGOU_GROK_API_KEY` 已配置
- [ ] 查看日志中的 "视频已下载到本地" 消息
